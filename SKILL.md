# 个股深度调研技能（券商体例整合版）

## 适用场景
- 用户要求调研/分析某只A股或港股
- 用户要求看某公司的投资逻辑、概念主线、未来几年预期
- 用户要求判断某股票估值是否合理（多维估值，不止PE）
- 用户上传年报/业绩公告/研报PDF要求精读或对比
- 用户要求看某公司业绩超预期判断、单季度趋势
- 用户提到"给我讲讲XX公司"、"XX怎么样"、"XX值不值得买"、"XX在炒什么"

## 七大核心目标（必须全部覆盖）
1. **公司在炒什么概念、什么逻辑** → 核心叙事章节
2. **行业地位如何** → 行业全景 + 竞争格局
3. **业绩如何** → 业绩快评（单季趋势+超预期判断+分项归因）
4. **未来几年预期** → 3年业绩展望 + 催化路径
5. **估值是否合理** → 多维估值矩阵 + 历史分位 + 三档情景隐含股价
6. **有没有跟什么公司合作** → 商业模式与产业链（上下游+大客户+合作披露）
7. **前景/卖方共识** → 卖方观点矩阵 + 催化剂时间线

## 输出物
结构化DOCX调研报告，11章 + 封面 + 免责声明，含多个对比表格

## 执行步骤

### Step 1：确认标的与材料
- 确认股票代码（如600667.SH / 002512.SZ）、全称、简称
- 询问用户是否提供材料：年报/业绩公告/研报PDF
  - 有 → 进入材料精读分支（Step 2A）
  - 无 → 进入纯搜索分支（Step 2B）

### Step 2A：材料精读分支（用户提供PDF时）

**A1. 年报精读**（如用户提供年报PDF）
提取：
- 主营业务构成（按业务/按地区/按产品）+ 收入占比
- 营收、归母净利润、扣非净利润、毛利率、净利率近3年趋势
- 经营性现金流、应收账款、存货周转
- 重大经营事项、子公司、关联方
- 风险提示与管理层讨论
- 研发投入、员工结构
- 重要客户/供应商披露（前5大或合作披露）

**A2. 业绩快评**（如用户提供业绩公告/快报/预告）
判断：
- **超预期/符合预期/低预期**（对照Wind一致预期或前期指引）
- 单季度环比/同比趋势（最近4个季度横向对比）
- 业绩驱动归因：量/价/成本/费用/一次性损益
- 与产业趋势是否吻合（行业Beta vs 公司Alpha）

**A3. 研报对比**（如用户提供1-10份券商研报PDF）
- 提取每份研报的：评级、目标价、盈利预测、核心观点
- 多份时生成**分歧矩阵**：盈利预测离散度、目标价区间、看多/看空分歧点
- 标注共识观点与分歧观点

### Step 2B：纯搜索分支（并行执行）
同时发起以下搜索：
1. `{公司名} {代码} 主营业务 业务结构 产业布局 2025 2026`
2. `{公司名} {代码} 年报 营收 净利润 财务数据 2025`
3. `{公司名} {代码} 行业地位 竞争对手 市场份额 龙头`
4. `{公司名} {代码} 市盈率 市净率 PE PB 估值 市值 最新`
5. `{公司名} {代码} 大客户 合作 战略合作 供应商 大订单`
6. `{公司名} {代码} 概念 板块 主题 炒作 催化`
7. `{公司名} {代码} 业绩 季报 单季 同比 环比 2026`
8. `{公司名} {代码} 券商 评级 目标价 研报`

### Step 3：行业全景搜索（来自equity-research:行业研究）
1. `{行业名} 市场规模 空间 复合增长 CAGR 2025 2026`
2. `{行业名} 产业链 上游 下游 关键环节`
3. `{行业名} 龙头 市占率 集中度 CR5`
4. `{行业名} 政策 行业趋势 技术演进`

### Step 4：估值对标搜索（来自equity-research:可比公司分析）
1. 确定2-4家同行业可比公司（业务相似+市值同档+地域可比）
2. 搜索可比公司多维估值：**PE-TTM / PB / PS / EV/EBITDA**
3. 搜索本公司历史估值分位（近3年/近5年PE中枢）
4. 搜索券商目标价、盈利预测（明年/后年EPS）

### Step 5：情绪与催化搜索
1. `{公司名} 近期 涨停 原因 利好 催化 2026`
2. `{公司名} 资金流向 龙虎榜 主力`
3. `{公司名} 板块联动 概念龙头`
4. **催化时间线**：未来3-12个月可能的股价催化点
   - 业绩披露窗口（季报/年报预约披露日）
   - 重大订单/产品发布预期
   - 行业政策窗口
   - 解禁/分红/分拆事件

### Step 6：数据核实
- 财务数据以年报/季报公告为准，不用券商预测值替代实际值
- PE计算方式：总市值 / 最近四个季度归母净利润（TTM）
- 可比公司PE数据必须用同期最新数据，不能用历史值
- 中财网常给中位/扣除后PE偏低，亿牛网/理杏仁给TTM最常用，统一口径
- 同行对比统一口径，混合口径即失真
- 如果搜索到的数据相互矛盾，以东方财富/同花顺/中财网等主流源为准

### Step 7：生成DOCX报告

报告结构（11章）：

```
封面：个股深度调研报告 + 公司名称代码 + 一句话定位 + 核心叙事一句话 + 报告日期

一、核心叙事与投资逻辑（800字内）
   - 公司在炒什么概念（一句话主线）
   - 为什么是现在炒（驱动事件/产业趋势）
   - 定位三分类：错杀→可介入 / 超跌反弹→短线 / 高位兑现→回避

二、公司概况
   - 代码/上市时间/实控人/核心子公司/总股本/市值/股价
   - 一句话主营业务定位

三、商业模式与产业链
   - 收入模式（卖什么/向谁卖/怎么卖）
   - 上游：关键原材料/供应商
   - 下游：主要客户/应用场景
   - 重要合作伙伴清单（战略合作/大订单/参股投资）

四、行业全景
   - 市场规模与CAGR（含口径与来源）
   - 产业链结构图（文字描述）
   - 关键技术节点与演进方向
   - 行业政策环境

五、行业地位与竞争格局
   - 公司在行业中的位置（龙头/二线/特色）
   - 市占率/排名
   - 竞争对手对比表格（4-5家）：营收/净利/毛利率/市占率/差异化

六、核心财务数据与业绩分析
   - 近3年财务总览表（营收/归母/扣非/毛利率/净利率/ROE）
   - 最新季度业绩快评：超预期/符合/低预期判断
   - 单季度趋势表（最近4-6个季度）
   - 业绩驱动归因（量/价/成本/费用/一次性）
   - 收入分项结构（按业务/按地区）

七、业绩展望（3年预测）
   - 2026E/2027E/2028E 营收与归母预测
   - 核心驱动假设（产能/单价/客户/渗透率）
   - 关键里程碑（订单/产能投产/产品发布）
   - 卖方一致预期 vs 自身判断的偏差

八、估值矩阵（多维）
   - 当前估值表：PE-TTM / PB / PS / EV-EBITDA / 市值
   - 历史估值分位：近3年/近5年中枢与百分位
   - 同行对比表：4-5家可比公司多维估值
   - 板块PE中枢对比（半导体等高成长板块单列）
   - **三档情景隐含股价**：
     * 保守：用同行最低PE × 保守EPS
     * 中性：用同行中位PE × 中性EPS
     * 乐观：用同行最高PE × 乐观EPS
   - 估值合理性结论（贵/合理/便宜，附三条理由）

九、卖方观点矩阵
   - 近期券商评级汇总表（机构/日期/评级/目标价/明年EPS预测）
   - 共识观点（≥3家提到的）
   - 主要分歧点（盈利预测离散度、看多看空分歧）

十、催化剂时间线（未来3-12个月）
   - 时间窗 + 事件 + 影响方向 + 确定性等级
   - 业绩披露 / 订单落地 / 政策窗口 / 产品发布 / 解禁分红

十一、情绪面与资金面分析
   - 近期股价异动复盘
   - 龙虎榜/北上资金/融资余额信号
   - 板块联动情况
   - 情绪驱动程度判断 + 退潮风险

十二、看多看空逻辑（各5-7条）
   - 看多逻辑：基本面/估值/情绪/催化
   - 看空逻辑：基本面/估值/情绪/风险

十三、综合评价（300字内）
   - 定位三分类判断
   - 核心矛盾点
   - 跟踪要点

免责声明 + 数据来源清单
```

### Step 8：输出与展示
- 文件保存到 outputs/ 目录，文件名 `{公司简称}_深度调研_{YYYYMMDD}.docx`
- 使用 present_files 展示给用户
- 简短口头总结核心结论（3-5句话）：
  - 一句话核心叙事
  - 估值贵/合理/便宜
  - 最大看点 + 最大风险
  - 跟踪建议

## 数据源优先级
1. 公司年报/季报/业绩公告（最权威）
2. 东方财富/同花顺/中财网实时数据
3. 财联社/证券时报/21世纪经济报道（行业报道）
4. 券商研报（观点参考，注意时效性）
5. 雪球/股吧（情绪参考，不作事实依据）
6. moomoo.com WebFetch经常超时，备选用亿牛网(eniu.com)、理杏仁、中财网

## 估值分析方法论（升级版）
1. **绝对估值**：当前PE/PB/PS/EV-EBITDA vs 公司历史中枢与分位 → 判断贵/便宜
2. **相对估值**：当前各维度估值 vs 同行业可比公司中位 → 判断溢价/折价
3. **机构观点**：券商目标价中位/区间 vs 当前股价 → 判断市场共识空间
4. **情绪溢价**：基本面能否支撑当前市值 → 判断情绪驱动程度
5. **板块环境**：同板块其他公司估值水平 → 个股问题还是板块普涨
6. **隐含股价**：三档情景EPS × 对应分位PE → 给出价格区间

## 报告风格要求
- 客观呈现数据，不做买卖建议
- 看多看空逻辑并列，让读者自己判断
- 明确标注数据来源和时效
- 对争议性数据或不一致之处如实说明
- 情绪面分析独立成章，不与基本面混淆
- 核心叙事章节用观察者视角，禁用"我们认为"

## 常见坑
- PE数据来源不同口径差异大（TTM vs 静态 vs 动态），必须标注口径
- 中财网常给中位/扣除后PE偏低，与亿牛网/理杏仁TTM不可混用
- 券商目标价可能过时（3-6个月前的研报），需注意时效
- 不要用预测利润算PE，只用已发布的实际利润
- moomoo.com WebFetch经常超时，备选用亿牛网/中财网
- 半导体等高成长板块PE普遍较高，不能简单用传统行业标准衡量
- 情绪面和基本面要分开讨论，不能混为一谈
- 业绩超预期判断必须对照Wind一致预期或前期指引，不能用主观感觉
- 行业市场规模数据多个口径（金额/出货量/装机量），统一口径
- 三档情景隐含股价不是预测，是情景推演，要说明假设
