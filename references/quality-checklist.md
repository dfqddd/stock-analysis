# 验证清单与风格要求

## 报告风格要求

- 客观呈现数据，不做买卖建议
- 看多看空逻辑并列，让读者自己判断
- 明确标注数据来源和时效
- 对争议性数据或不一致之处如实说明
- 情绪面分析独立成章，不与基本面混淆
- 核心叙事章节用观察者视角，禁用"我们认为"

## 完成度检查清单

- [ ] 核心叙事章节明确写出"在炒什么+为什么是现在"
- [ ] 商业模式章节有具体合作伙伴/大客户披露
- [ ] 行业全景有市场规模数据（带来源和口径）
- [ ] 业绩部分包含单季度趋势 + 超预期判断
- [ ] 业绩展望有3年预测 + 核心假设
- [ ] 估值矩阵覆盖PE/PB/PS/EV-EBITDA四维
- [ ] 给出三档情景隐含股价
- [ ] 卖方观点矩阵≥3家券商
- [ ] 催化剂时间线≥3个具体事件，带时间窗
- [ ] 所有财务数据有明确来源（年报/季报/公告）
- [ ] PE/PB计算口径已标注，同行对比口径统一
- [ ] 可比公司数据为同期最新
- [ ] 情绪面有具体事件和数据支撑
- [ ] 看多看空逻辑平衡，无明显偏向
- [ ] DOCX格式无乱码，表格对齐

## 其他注意事项

- moomoo.com WebFetch经常超时，备选用亿牛网/中财网
- 情绪面和基本面要分开讨论，不能混为一谈
- 三档情景隐含股价不是预测，是情景推演，要说明假设
