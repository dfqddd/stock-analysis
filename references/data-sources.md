# 数据源与核实规则

## 数据源优先级

1. 公司年报/季报/业绩公告（最权威）
2. 东方财富/同花顺/中财网实时数据
3. 财联社/证券时报/21世纪经济报道（行业报道）
4. 券商研报（观点参考，注意时效性）
5. 雪球/股吧（情绪参考，不作事实依据）
6. moomoo.com WebFetch经常超时，备选用亿牛网(eniu.com)、理杏仁、中财网

## 数据核实规则

- 财务数据以年报/季报公告为准，不用券商预测值替代实际值
- PE计算方式：总市值 / 最近四个季度归母净利润（TTM）
- 可比公司PE数据必须用同期最新数据，不能用历史值
- 中财网常给中位/扣除后PE偏低，亿牛网/理杏仁给TTM最常用，统一口径
- 同行对比统一口径，混合口径即失真
- 如果搜索到的数据相互矛盾，以东方财富/同花顺/中财网等主流源为准
