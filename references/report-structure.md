# 报告结构（13章）

```
封面：个股深度调研报告 + 公司名称代码 + 一句话定位 + 核心叙事一句话 + 报告日期

一、核心叙事与投资逻辑（800字内）
   - 公司在炒什么概念（一句话主线）
   - 为什么是现在炒（驱动事件/产业趋势）
   - 定位三分类：错杀→可介入 / 超跌反弹→短线 / 高位兑现→回避

二、公司概况
   - 代码/上市时间/实控人/核心子公司/总股本/市值/股价
   - 一句话主营业务定位

三、商业模式与产业链
   - 收入模式（卖什么/向谁卖/怎么卖）
   - 上游：关键原材料/供应商
   - 下游：主要客户/应用场景
   - 重要合作伙伴清单（战略合作/大订单/参股投资）

四、行业全景
   - 市场规模与CAGR（含口径与来源）
   - 产业链结构图（文字描述）
   - 关键技术节点与演进方向
   - 行业政策环境

五、行业地位与竞争格局
   - 公司在行业中的位置（龙头/二线/特色）
   - 市占率/排名
   - 竞争对手对比表格（4-5家）：营收/净利/毛利率/市占率/差异化

六、核心财务数据与业绩分析
   - 近3年财务总览表（营收/归母/扣非/毛利率/净利率/ROE）
   - 最新季度业绩快评：超预期/符合/低预期判断
   - 单季度趋势表（最近4-6个季度）
   - 业绩驱动归因（量/价/成本/费用/一次性）
   - 收入分项结构（按业务/按地区）

七、业绩展望（3年预测）
   - 2026E/2027E/2028E 营收与归母预测
   - 核心驱动假设（产能/单价/客户/渗透率）
   - 关键里程碑（订单/产能投产/产品发布）
   - 卖方一致预期 vs 自身判断的偏差

八、估值矩阵（多维）
   - 当前估值表：PE-TTM / PB / PS / EV-EBITDA / 市值
   - 历史估值分位：近3年/近5年中枢与百分位
   - 同行对比表：4-5家可比公司多维估值
   - 板块PE中枢对比（半导体等高成长板块单列）
   - 三档情景隐含股价：
     * 保守：用同行最低PE × 保守EPS
     * 中性：用同行中位PE × 中性EPS
     * 乐观：用同行最高PE × 乐观EPS
   - 估值合理性结论（贵/合理/便宜，附三条理由）

九、卖方观点矩阵
   - 近期券商评级汇总表（机构/日期/评级/目标价/明年EPS预测）
   - 共识观点（≥3家提到的）
   - 主要分歧点（盈利预测离散度、看多看空分歧）

十、催化剂时间线（未来3-12个月）
   - 时间窗 + 事件 + 影响方向 + 确定性等级
   - 业绩披露 / 订单落地 / 政策窗口 / 产品发布 / 解禁分红

十一、情绪面与资金面分析
   - 近期股价异动复盘
   - 龙虎榜/北上资金/融资余额信号
   - 板块联动情况
   - 情绪驱动程度判断 + 退潮风险

十二、看多看空逻辑（各5-7条）
   - 看多逻辑：基本面/估值/情绪/催化
   - 看空逻辑：基本面/估值/情绪/风险

十三、综合评价（300字内）
   - 定位三分类判断
   - 核心矛盾点
   - 跟踪要点

免责声明 + 数据来源清单
```
