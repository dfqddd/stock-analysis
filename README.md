# stock-analysis

个股调研分析工具库。配合 QoderWork 的 `stock-research` 技能使用。

## 目录结构

```
SKILL.md          # 技能定义文件（调研流程/方法论/验证清单）
templates/        # DOCX 生成脚本模板
examples/         # 已完成的调研报告示例
```

## 使用方式

在 QoderWork 中说"调研一下 XX 股票"即可自动触发 `stock-research` 技能，按照 SKILL.md 中的流程执行。
